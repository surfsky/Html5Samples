🚀 浏览器智能缓存

Back/Forward Cache (BFCache) 是现代浏览器的一项强大功能，它可以将整个页面（包括JavaScript状态）保存在内存中，当用户使用浏览器的前进/后退按钮时，页面可以瞬间恢复。

🔧 BFCache 优化技巧

  • 避免 unload 事件: 使用 pagehide 和 pageshow 事件
  • 清理定时器: 在页面隐藏时暂停，显示时恢复
  • 避免持久连接: WebSocket、EventSource等会阻止缓存
  • 合理使用Cache-Control: 不要设置 no-store
// BFCache 友好的事件处理 window.addEventListener('pageshow', (event) => { if (event.persisted) { console.log('页面从BFCache恢复'); // 恢复定时器、动画等 } }); window.addEventListener('pagehide', (event) => { if (event.persisted) { console.log('页面进入BFCache'); // 暂停定时器、清理资源 } });
